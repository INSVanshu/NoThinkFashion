## ðŸŽ¥ PRODUCTION NOTES

### Recording Setup
- **Software**: OBS Studio or Loom
- **Resolution**: 1920x1080 (1080p minimum)
- **Frame rate**: 30fps or 60fps
- **Format**: MP4 (H.264 codec)

### Screen Recording Tips
1. **Close unnecessary tabs** - clean browser
2. **Hide bookmarks bar** - cleaner look
3. **Use incognito mode** - no autofill suggestions
4. **Prepare sample data** - have 5-6 clothing photos ready
5. **Test run first** - practice the demo flow

### Audio Recording
- **Microphone**: Use good USB mic or headset
- **Environment**: Quiet room, no echo
- **Script**: Read naturally, don't sound robotic
- **Pacing**: Speak clearly, not too fast
- **Energy**: Be enthusiastic but not over-the-top

### Editing
- **Software**: DaVinci Resolve (free), iMovie, or Adobe Premiere
- **Transitions**: Keep simple (crossfades, cuts)
- **Music**: Upbeat, modern (check royalty-free sites)
  - Recommended: YouTube Audio Library, Epidemic Sound
  - Keep volume LOW (shouldn't overpower voice)
- **Text overlays**: Use clean sans-serif font (Space Grotesk to match brand)
- **Color grading**: Enhance purple/blue tones to match app

### Before/After Checklist
**Before Recording:**
- [ ] Test API key works
- [ ] Prepare clothing photos (5-6 varied items)
- [ ] Clear browser cache
- [ ] Close unnecessary apps
- [ ] Test microphone levels
- [ ] Have script visible (second monitor or printed)

**After Recording:**
- [ ] Watch full video (check for mistakes)
- [ ] Add background music (subtle)
- [ ] Add on-screen text/URLs
- [ ] Export at 1080p
- [ ] Upload to YouTube (unlisted)
- [ ] Add to Devpost submission
